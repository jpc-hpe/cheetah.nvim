1. Introduction                                        *cheetah-introduction*

cheetah is an experimental Lua plugin for Neovim. This plugin serves as a
sandbox for experimentation with Neovim's Lua API.

==============================================================================
2. Installation                                        *cheetah-sinstallation*

You can install this plugin using your favorite plugin manager:

* With packer.nvim:
>
    use {
      'jpc-hpe/testplugin.nvim',
      config = function()
        require('cheetah').setup{}
      end
    }
<

* With lazy.nvim:
>
    {
      'jpc-hpe/testplugin.nvim',
      config = function()
        require('cheetah').setup{}
      end
    }
<

* With vim-plug:
>
    Plug 'jpc-hpe/testplugin.nvim'
<
    After installation, include in your configuration:
>
    lua require('cheetah').setup{}
<

==============================================================================
3. Configuration                                      *cheetah-configuration*

You can configure the plugin by passing options to the setup function:
>
    require('cheetah').setup({
      enabled = true,
      -- Add more configuration options as needed
    })
<
